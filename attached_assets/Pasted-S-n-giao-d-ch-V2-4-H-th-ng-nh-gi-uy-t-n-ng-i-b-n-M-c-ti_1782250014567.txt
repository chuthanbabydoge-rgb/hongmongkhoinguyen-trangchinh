Mô hình miền

Uy tín người bán

{
userId:string,

score:number,

level:string,

totalSales:number,

totalVolume:number,

positiveRatings:number,

negativeRatings:number,

createdAt:string,

updatedAt:string

}

Các cấp độ

0-19 Đồng

20-49 Bạc

50-99 Vàng

100-199 Bạch kim

200+ Kim cương

Mô hình xếp hạng

Sau khi mua:

người mua có thể xếp hạng Người bán

Xếp hạng:

+1 tích cực

-1 tiêu cực

API

POST /api/marketplace/reputation/rate

{
buyerId,
sellerId,
transactionId,
rating
}

GET /api/marketplace/reputation/:userId

GET /api/marketplace/reputation/top-sellers

Ví dụ phản hồi

{
"ok":true,

"data":{
"userId":"abc",

"score":74,

"level":"Gold",

"totalSales":25,

"totalVolume":180000,

"positiveRatings":22,

"negativeRatings":3

}
}

Công thức tính điểm

score =

(totalSales * 2)

+
(positiveRatings * 5)
---------------------

(negativeRatings * 10)

Tự động tính toán lại sau mỗi lần đánh giá.
