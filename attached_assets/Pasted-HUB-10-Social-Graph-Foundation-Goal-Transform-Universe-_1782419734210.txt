CANCELLED

========================================
2. Friend Requests
==================

Endpoints:

POST   /api/social/friends/request
POST   /api/social/friends/:id/accept
POST   /api/social/friends/:id/decline

GET    /api/social/friends
GET    /api/social/friends/pending
GET    /api/social/friends/sent

Rules:

* cannot friend self
* cannot duplicate request
* accepting creates FRIEND relationship both ways
* notification generated automatically

========================================
3. Follow System
================

Endpoints:

POST   /api/social/follow/:userId
DELETE /api/social/follow/:userId

GET    /api/social/followers
GET    /api/social/following

Rules:

* one-way relationship
* notification generated on follow

========================================
4. Public Profiles
==================

Endpoints:

GET /api/social/profile/:userId
GET /api/social/profile/me

Response:

{
userId,
displayName,
avatar,
reputation,
level,
achievements,
followers,
following,
friends,
joinDate
}

Data sources:

* Reputation Service
* Achievement Service
* Account Profile
* Social Graph
